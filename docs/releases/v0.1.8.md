# OOTP Milestone Tracker v0.1.8

**릴리즈 일자:** 2026-06-16  
**유형:** 패치 (UI 레이아웃)

v0.1.7 대비 **기능 추가 없음**. 화면 배치를 조정해 기본 창 크기를 줄였습니다. **글씨 크기는 그대로**입니다.

---

## 변경 사항

### 창 크기
- 메인 창 기본 크기: **1100×720 → 780×520** (약 70%)

### 선수 기록
- 타격 / 투구 / 마일스톤을 **탭**으로 분리 (세로 스크롤 감소)
- 선수 목록 **폭 제한**, import·필터 컨트롤 **2줄** 배치

### 마일스톤 기록
- 필터(대상·팀·scope·시즌)와 액션 버튼 **2줄** 분리

### 대시보드
- 리그 상태와 가져오기 버튼 **한 줄** 배치, 중복 제목 제거

### 설정
- **2열** 배치 (세이브·리그 / 추적·한글·마일스톤·DB)
- 긴 설명 문구는 **툴팁**으로 이동
