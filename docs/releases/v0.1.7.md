# OOTP Milestone Tracker v0.1.7

**릴리즈 일자:** 2026-06-16  
**유형:** 첫 공개 릴리즈 (First Release)

OOTP(Out of the Park Baseball) 세이브에서 선수·팀 마일스톤을 추적하고, 달성을 기록하며, 통산·시즌 목표 달성 가능성을 미리 살펴볼 수 있는 Windows 데스크톱 앱입니다.

---

## 하이라이트

- 박스스코어 HTML import → 경기·시즌·통산·팀 마일스톤 **자동 감지**
- **266+** 마일스톤 기준 (`milestones.csv`, 앱 내 편집 가능)
- 관심 구단만 골라 보는 **추적 팀** 필터 (MLB 30팀 + **커스텀 팀**)
- stats export **초기값 import**로 통산·과거 시즌 baseline 구축
- **마일스톤 예측** — 임박 항목 강조, 남은 수치 기준 추적 시작
- **수동 입력** — 수상·이적·부상 포함; DB에 없는 선수도 풀 네임으로 등록 가능
- 세이브(리그)마다 **별도 DB** — 리그 전환 시 데이터 분리

---

## 포함 기능

### 마일스톤 기록
- 박스스코어 import 후 자동 마일스톤 체크 (개인·팀, 경기·시즌·통산)
- 마일스톤 기록 탭: 이력 조회, 필터, 수동 입력, 시즌 비율 마일스톤 일괄 판정
- 연속 기록(streak) 마일스톤 — import 연동, CSV 내보내기
- 수동 입력 통합 팝업
  - **마일스톤 / 수상** — 개인·팀
  - **이적** — FA·연장·트레이드·선수 구매 (합류/이탈 선수, 팀 선택)
  - **부상** — 부상명·기간·소속팀
- **수동 선수 등록** — 추적 팀·stats·박스스코어에 아직 없는 선수도 풀 네임(예: `Dong-ju Moon`)으로 기록; 목록에는 `D. Moon` 형식 표시, 이후 import 시 **자동 병합**

### 마일스톤 예측
- 통산·시즌 마일스톤 달성 가능성 표시
- 목표까지 **남은 수치**가 일정 비율 이하일 때 추적 시작 (기본: 목표의 15%)
- `near_n` 임박 항목 **행 배경색** 강조

### 선수 기록
- 추적 팀 선수 목록, 시즌·통산 전환
- 박스스코어 있을 때 경기별 드릴다운
- **stats 파일만으로도** 과거 시즌·통산 조회 (박스스코어 없을 때 init fallback)
- 커스텀 MLB 팀(확장팀 등) 추적 — stats export의 팀 소속 정보 연동

### 데이터 import
| OOTP export | 용도 |
|-------------|------|
| `news/html/box_scores/*.html` | 경기 기록, 실시간 마일스톤 |
| `player_batting_stats.txt` / `player_pitching_stats.txt` | 통산·과거 시즌 초기값 |
| `mlb_rosters` / `kbo_rosters` | 레이팅 편집 전용 (통계와 별도) |

- **초기값 설정** 탭: 최초 import, 시즌 갱신(refresh), 시즌 중(mid_season) 모드
- MLB 전용 import 옵션 (WBC·KBO 등 제외 가능)

### 그 외
- **대시보드** — 최근 마일스톤·요약
- **레이팅 편집** — MLB+KBO 로스터 조건별 일괄 수정, `mod_*_rosters.txt` 저장
- **한글 이름** — CSV 매핑·pending 큐·설정 UI
- **설정** — OOTP 경로, 추적 팀, 마일스톤 기준 편집, **현재 세이브 DB 초기화**

## 시작하기

1. `ootp_milestone_tracker_v0.1.7.zip` 압축 해제 후 `ootp_milestone_tracker.exe` 실행
2. **설정** 탭에서 OOTP 세이브의 `import_export` 폴더 경로·현재 시즌·추적 팀 지정  
   - 확장팀 등은 **팀 수동 추가**로 약칭·팀 이름 등록
3. **초기값 설정**에서 batting/pitching stats 파일 import
4. **선수 기록** 또는 **마일스톤 기록**에서 박스스코어 HTML import
5. **마일스톤 기록 / 예측** 탭에서 결과 확인

> 사용자 데이터(DB·설정·한글 CSV)는 `%APPDATA%\OOTP_Milestone_Tracker\`에 저장됩니다. exe를 교체해도 기록이 유지됩니다.

## 알아두면 좋은 점

- **신생팀·확장팀:** stats export에 해당 팀(`SY` 등) 행이 없으면 통계 탭 필터에 선수가 안 보일 수 있습니다. 그동안 **수동 입력**으로 선수를 등록해 마일스톤을 남기고, stats·박스스코어가 쌓이면 자동 연결됩니다.
- **로스터 파일**(`mlb_rosters` / `kbo_rosters`)은 레이팅 편집용이며, 통계·마일스톤 DB와는 연동되지 않습니다.
- **경기별 상세·실시간 시즌 누적**은 박스스코어 import가 필요합니다.
