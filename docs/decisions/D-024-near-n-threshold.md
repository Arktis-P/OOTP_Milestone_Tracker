# D-024: 임박 마일스톤 기준 (`near_n`)

## 상태

확정 (2026-06-12)

## 맥락

통산 마일스톤 예측 탭에는 달성률이 높은 항목이 많아, "곧 달성할" 기록을 한눈에 구분하기 어렵다.
대시보드에서도 임박 항목을 별도로 보여줄 필요가 있다.

## 결정

- `milestones.csv`에 `near_n` 컬럼을 추가한다.
- `remaining <= near_n`이면 **임박**으로 표시한다 (`🔥 임박` 배지).
- `near_n` 미지정 시 기본값: `int(threshold * 0.05)` (5%, 정수 내림).
- 페이스·시즌 잔여 경기 추정은 **사용하지 않는다** — 절대 수치 비교만.
- 예측 목록 정렬: `is_near` 내림차순 → `progress_pct` 내림차순.

## 대안

| 대안 | 기각 이유 |
|------|-----------|
| 달성률 % 기준 (예: 95% 이상) | 홈런 500개(20 남음) vs 안타 3000개(150 남음)에서 체감이 다름 |
| 시즌 페이스 기반 임박 | 구현·검증 복잡, 시즌 초 데이터 부족 |
| 전역 고정 near_n | 마일스톤 종류별 스케일 차이 반영 불가 |

## 결과

- 예측 탭·대시보드에서 career scope 임박 항목 강조.
- 마일스톤별로 Excel에서 `near_n`을 직접 조정 가능.
