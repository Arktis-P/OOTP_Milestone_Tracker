# Changelog

## [0.1.4] - 2026-06-16

### Changed
- 통산 마일스톤 예측 `track_from`을 남은 수치 기준으로 변경 (기본: 목표의 15% 남음)

- SQLite 집계 및 마일스톤 체크/예측 기본 로직
- PyQt6 탭 기반 GUI (마일스톤, 기록, 예측, 로스터)
- PyInstaller 빌드 스크립트 (`build.py`)
